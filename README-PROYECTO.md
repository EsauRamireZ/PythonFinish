# Documentación rápida

## IDE
- Visual Studio Code

## Framework
- Node.js
- Express.js
- EJS
- Fetch API
- DOM nativo
- SQL Server con mssql

## Hosting
- Compatible con Render o cualquier hosting Node.js
- No se modificaron tus cadenas sensibles de Render, base de datos ni reCAPTCHA

## Código fuente
- Login con reCAPTCHA
- JWT por cookie httpOnly
- CRUD de Perfil, Módulo, Permisos Perfil y Usuario
- Paginación de 5 filas
- Breadcrumbs en todos los módulos
- Módulos estáticos Principal 1.1, 1.2, 2.1 y 2.2
- Imagen de usuario
- Página de error personalizada
